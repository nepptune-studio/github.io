6시간 - 기술 선택

  • 호스팅은 Github Pages. 저렴, 용이.
  • Jekyll + Ruby를 사용하지 않음. 학습 코스트 큼, 정적. Ruby 비선호.
  • html + css + js로 시작하기로. 웹개발 해커톤 경험 1회 (버튼 인풋과 음악 재생 기능이 있는 간단한 웹사이트)

  • 2024년 웹개발 현업에 쓰이는 TypeScript + React/Next.js? 향후 마이그레이션 가능성 열어둠.

  • 기술 블로그 코드 작성 중 아이디어: markdown 파일에 자유롭게 기술, 추후 정리.
